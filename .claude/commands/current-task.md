# /current-task

Show the current task being worked on.

## Usage

```
/current-task
```

## Description

The `/current-task` command displays information about the task that is currently being executed, including its ID and description.

## Example

```
/current-task
```

This will show:
- Task ID
- Task Description
- Status

## Related Commands

- `/speckit.tasks` - See all tasks
- `/help` - See all available commands
