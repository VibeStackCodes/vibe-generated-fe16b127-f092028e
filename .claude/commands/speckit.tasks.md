# /speckit.tasks

Break down the implementation plan into specific, actionable tasks.

## Usage

Use this command to see all tasks that need to be executed to complete the project.

## Description

The `/speckit.tasks` command lists all tasks that have been generated from the PRD. These tasks are executed sequentially to implement the project.

## Example

```
/speckit.tasks
```

This will display a numbered list of all tasks with their descriptions.

## Related Commands

- `/speckit.plan` - Generate the implementation plan first
- `/current-task` - See which task is currently being worked on
