# /help

Show available slash commands.

## Usage

```
/help
```

## Description

The `/help` command displays a list of all available slash commands and their descriptions.

## Available Commands

- **/speckit.plan** - Generate an implementation plan from the PRD
- **/speckit.tasks** - Break down the plan into specific tasks
- **/speckit.spec <name> [content]** - Create or update specifications
- **/current-task** - Show the current task being worked on
- **/help** - Show this help message

## Getting Started

1. First, use `/speckit.plan` to create a high-level implementation plan
2. Then use `/speckit.tasks` to see all tasks
3. Execute tasks sequentially
4. Use `/current-task` to track progress
5. Update specifications as needed with `/speckit.spec`
